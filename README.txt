
# ❤️ Irrna Birthday Website

## Features
- Password lock screen
- Typing animation
- Floating hearts
- Romantic pastel Apple aesthetic
- Photo gallery
- Background music
- Countdown timer
- Mini game
- ZUS gift section

## Deploy to Vercel 🚀

1. Upload this folder to GitHub
2. Go to https://vercel.com
3. Import GitHub repository
4. Click Deploy

## Customize
- Replace music in HTML
- Add your ZUS coupon link
- Change countdown date in script.js
